ChatNow : help
ChatNow is a Chatting platform built using NodeJS. Using this platform anyone can create a chatroom and share it with someone else to chat with or can join an already created chatroom using invite link or manually entering the chat ID. Chatrooms inactive for more than 10 minutes are deleted automatically along with all the chats and userdata.

Create : help
When using this service , users can create a chatroom by just entering their name . Through this functionality, a 16 characters chat ID is generated randomly. This ID is the access to the chatroom. Anyone with this ID can join the Chat.

If Any person join the chatroom after wipedown, then a new chatroom is created with the Entered Chat ID as it does not exist.

Join : help
The Join Portal gives user the ability to join a previously created chatroom or to create a new one in case it does not exist. Users can join a chatroom using the Invite link or by entering the chat ID of an already created by entering their name.

In case the requested chatroom is not created or it has been deleted due to inactivity , A new chatroom is created with the ID entered by User.
